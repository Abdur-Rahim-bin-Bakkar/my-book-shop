## ✨ Features  

- 📖 Browse a collection of popular books  
- 🔍 View detailed book information  
- ❤️ Add books to Wishlist  
- 📚 Mark books as Read  
- 📊 Clean and responsive UI design  
- ⚡ Fast performance with React  

---

## 🛠️ Technologies Used  

- ⚛️ React  
- 🎨 Tailwind CSS  
- 📦 React Router  
- 📊 Recharts (for charts)  
- 🗂️ JSON Data
